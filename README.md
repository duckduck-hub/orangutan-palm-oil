# Orangutan Palm Oil Project

## Project Description
This project is dedicated to the research and advocacy for the protection of orangutans from palm oil cultivation.

## Install Instructions
1. Clone the repository: `git clone https://github.com/duckduck-hub/orangutan-palm-oil.git`
2. Navigate into the project directory: `cd orangutan-palm-oil`
3. Install dependencies (if any): `npm install` (or your package manager of choice)

## Usage
To run the project, use the following command: `npm start`.

## Enabling GitHub Pages
To enable GitHub Pages for this project:
1. Go to the repository on GitHub.
2. Click on **Settings**.
3. Scroll down to the **Pages** section.
4. Under **Source**, select the branch you want to publish (usually `main`) and click **Save**.
5. Your site will be published at `https://<username>.github.io/orangutan-palm-oil/`.
